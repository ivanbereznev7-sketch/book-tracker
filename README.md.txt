Автор: Березнев Иван Викторович

Дата: Декабрь 2024

 Описание программы
Book Tracker — это графическое приложение для ведения личного каталога прочитанных книг. Программа позволяет:

 Добавлять книги с указанием названия, автора, жанра и количества страниц

 Просматривать список всех добавленных книг в удобной таблице

 Фильтровать книги по жанру и количеству страниц

 Автоматически сохранять данные в JSON-файл

 Удалять книги из списка

 Отслеживать общее количество книг в библиотеке

Особенности
 Интуитивно понятный графический интерфейс

 Валидация всех вводимых данных

 Автоматическое сохранение в JSON

 Мощная система фильтрации

 Информативная статусная строка

 Всплывающие подсказки для всех полей

 Системные требования
Python версии 3.8 или выше

Стандартные библиотеки Python (tkinter, json, datetime)

Откройте терминал в этой папке

 Руководство пользователя
Главное окно программы
При запуске открывается окно, разделённое на две основные части:

Левая панель - форма добавления книги
Поле	Описание	Пример
Название книги	Название произведения	"Война и мир"
Автор	ФИО автора	"Лев Толстой"
Жанр	Выбор из выпадающего списка	"Роман"
Количество страниц	Целое положительное число	1300
Правая панель - список книг и фильтрация
Таблица книг: показывает все добавленные книги

Фильтр по жанру: показывает книги только выбранного жанра

Фильтр по страницам: показывает книги с количеством страниц больше указанного

Кнопка удаления: удаляет выбранную книгу

 Примеры использования
Пример 1: Добавление книги
Заполните поля формы:

Название: "Мастер и Маргарита"

Автор: "Михаил Булгаков"

Жанр: "Роман"

Страницы: "400"

Нажмите "Добавить книгу"

Книга появится в таблице справа

Пример 2: Фильтрация книг
Фильтр по жанру "Фантастика":

Выберите в выпадающем списке "Фантастика"

Нажмите "Применить фильтр"

В таблице останутся только книги жанра "Фантастика"

Фильтр по страницам (> 300):

Введите в поле "300"

Нажмите "Применить фильтр"

Покажутся книги с более чем 300 страницами

Комбинированный фильтр:

Выберите жанр "Детектив"

Введите минимальное количество страниц "200"

Нажмите "Применить фильтр"

Будут показаны детективы с количеством страниц больше 200

Пример 3: Удаление книги
Кликните на книгу в таблице (строка выделится)

Нажмите "Удалить выбранную книгу"

Подтвердите удаление в диалоговом окне

 Возможные проблемы и решения
Проблема: Приложение не запускается
Решение: Убедитесь, что установлен Python 3.8 или выше:

bash
python --version
Проблема: Ошибка "No module named tkinter"
Решение: Tkinter обычно идёт в комплекте с Python. Если нет:

Ubuntu/Debian: sudo apt-get install python3-tk

Windows: Переустановите Python, отметив опцию "tcl/tk"

Проблема: Не сохраняются книги
Решение: Проверьте права на запись в папке с программой
